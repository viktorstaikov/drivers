Drivers Background Check system

Used technologies: Python, Flask (web framework for Python), Peewee (orm), SQLite (database), AngularJS (fron-end), Bootstrap (styling) and Bower (managing front-end dependencies).

This system supports the following operations:

  • Adding a new driver.
  • Listing all drivers in the system with all the information about them, including the status of their background check.
  • Deleting a certain driver.
  • Exporting in CSV format the data for 'unchecked' drivers.
  • Importing in CSV format the data for background checks of drivers. If a driver no longer (or has never existed, but somehow is presented), nothing hapens.
